# Voynich Manuscript: Interlinear Translation

**EVA Transcription with Croatian Reading**

Using the ZFD (Zuger Functional Decipherment) three-layer key:
- Operators: q/qo -> ko, ch -> h, sh -> š
- Gallows (medial): k -> st, t -> tr, f -> pr, p -> pl
- Suffixes: y -> i, n -> n, r -> r, etc.
